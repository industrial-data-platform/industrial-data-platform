# ADR Archive

Дата: 2026-05-10
Статус: historical archive

ADRs are no longer default source-of-truth for current system behavior. They are
kept here as historical rationale: why a decision was made, which alternatives
were rejected, and what trade-offs were accepted at the time.

For active architecture navigation, start with
[`../decisions.md`](../decisions.md), then use:

- [`../current-state.md`](../current-state.md) for the working system snapshot.
- [`../glossary.md`](../glossary.md) for terminology.
- [`../../contracts/README.md`](../../contracts/README.md) for contract details.
- [`../../../arch/README.md`](../../../arch/README.md) for LikeC4 model entrypoints.

The full ADR files live in [`archive/`](archive/). If an archived ADR disagrees
with `docs/contracts/`, the contract document wins.
